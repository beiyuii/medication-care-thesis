## 图 3-3 系统业务时序图

```mermaid
sequenceDiagram
    autonumber
    actor User as 老年人用户
    participant Frontend as Vue3 前端
    participant Backend as Spring Boot 后端
    participant DB as MySQL 数据库
    participant Flask as Flask 算法服务

    User->>Frontend: 登录系统
    Frontend->>Backend: POST /auth/login
    Backend->>DB: 校验账户与角色
    DB-->>Backend: 返回用户信息
    Backend-->>Frontend: token + role
    Frontend-->>User: 跳转首页

    User->>Frontend: 创建用药计划
    Frontend->>Backend: POST /schedules
    Backend->>DB: 写入计划数据
    DB-->>Backend: 返回计划 ID
    Backend-->>Frontend: 创建成功

    loop 到达提醒时间窗
        Frontend-->>User: 弹出提醒并引导开启检测
        User->>Frontend: 开启检测并上传视频
        Frontend->>Backend: POST /api/detection-jobs
        Backend->>DB: 创建检测任务并更新提醒实例
        Backend->>Flask: /v1/detections/video/predict
        Flask->>Flask: 抽帧、目标检测、动作识别
        Flask-->>Backend: 返回 confirmed / suspected / abnormal
        Backend->>DB: 写入服药事件并更新状态
        alt 检测异常
            Backend->>DB: 生成异常告警
        end
        Backend-->>Frontend: 返回检测结果
    end

    alt 结果为 suspected 或 abnormal
        User->>Frontend: 手动确认已服药
        Frontend->>Backend: POST /reminder-instances/{id}/confirm
        Backend->>DB: 更新事件与实例状态为 confirmed
        Backend-->>Frontend: 返回确认成功
    end

    Frontend-->>User: 展示最终服药状态与历史记录
```

## 图 4-3 用药计划模块时序图

```mermaid
sequenceDiagram
    autonumber
    actor User as 老年人用户
    participant Frontend as Vue3 前端
    participant Controller as ScheduleController
    participant Mapper as ScheduleMapper
    participant DB as MySQL

    User->>Frontend: 填写药品名称、剂量、频次、时间窗
    Frontend->>Controller: POST /api/schedules
    Controller->>Mapper: insert(schedule)
    Mapper->>DB: 写入 schedules 表
    DB-->>Mapper: 返回主键 ID
    Mapper-->>Controller: 返回新增计划
    Controller-->>Frontend: 返回创建成功结果
    Frontend-->>User: 刷新计划列表

    User->>Frontend: 切换计划启停状态
    Frontend->>Controller: POST /api/schedules/{id}/toggle
    Controller->>Mapper: selectById(id)
    Mapper->>DB: 查询计划
    DB-->>Mapper: 返回当前状态
    Controller->>Mapper: updateById(schedule)
    Mapper->>DB: 更新状态 enabled / disabled
    DB-->>Mapper: 返回更新结果
    Controller-->>Frontend: 返回最新计划对象
```

## 图 4-4 目标检测模块时序图

```mermaid
sequenceDiagram
    autonumber
    actor User as 老年人用户
    participant Frontend as Vue3 前端
    participant Spring as DetectionJobService
    participant DB as MySQL
    participant Flask as Flask /video/predict
    participant Algo as YOLOv8 + MediaPipe

    User->>Frontend: 开启摄像头并录制视频
    Frontend->>Spring: 上传 videoFile 与 patientId
    Spring->>DB: 创建 detection_jobs 记录
    Spring->>DB: 更新 reminder_instances 为 detecting
    Spring->>Flask: 转发 multipart 视频请求
    Flask->>Flask: 提取视频元数据与关键帧
    Flask->>Algo: 执行药品目标检测
    Algo-->>Flask: 返回目标框与置信度
    Flask->>Algo: 执行手口距离动作识别
    Algo-->>Flask: 返回动作时间线
    Flask->>Flask: 聚合为 confirmed / suspected / abnormal
    Flask-->>Spring: 返回检测结果 JSON
    Spring->>DB: 更新 detection_jobs 结果
    Spring->>DB: 写入 intake_events
    alt abnormal
        Spring->>DB: 生成 alerts 告警
    else confirmed
        Spring->>DB: 关闭已有告警并更新实例状态
    end
    Spring-->>Frontend: 返回检测状态
    Frontend-->>User: 展示检测结果并允许人工确认
```
