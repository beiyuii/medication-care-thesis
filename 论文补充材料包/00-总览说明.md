# 论文补充材料包总览

本文件夹用于集中整理《基于目标检测的老年人用药提醒与管理系统的设计与实现》当前可补入论文的图表、数据库表结构、正文代码片段和插入建议，便于统一查看与后续排版。

## 文件结构

1. `01-图表材料/现有图表与截图总览.md`
   汇总当前仓库中已经存在的页面截图、E-R 图，以及仍需补拍或重导出的图表项。
2. `01-图表材料/补充图表源文件.md`
   提供缺失图表的 Mermaid 源码，包括业务时序图、系统技术架构图、功能层次结构图和两个核心模块时序图。
3. `02-数据库表结构/数据库物理结构设计表.md`
   按论文要求整理的数据库物理结构设计表，包含字段名称、数据类型、键、允许空和作用说明。
4. `03-正文代码片段/正文代码片段汇总.md`
   提炼后的正文可用代码片段，分别对应前端、Spring Boot 业务层和 Flask 算法调用。
5. `04-插入位置与说明/论文插入建议.md`
   标明各图、表、代码片段建议放入的章节位置、建议图题表题和配套说明文字。
