# 视频检测测试页面使用说明

## 快速开始

### 1. 启动服务

```bash
cd flask-project
python -m app.main
```

或者：

```bash
cd flask-project
flask --app app.main run
```

服务将在 `http://localhost:8000` 启动。

### 2. 访问测试页面

在浏览器中打开以下任一地址：

- `http://localhost:8000/`
- `http://localhost:8000/test`

### 3. 使用测试页面

1. **填写表单信息**：
   - 患者ID（必需）
   - 计划ID（必需）
   - 时间戳（必需，默认当前时间）
   - 摄像头ID（可选）
   - 模型版本（可选）
   - 采样率（可选，默认30帧/秒）
   - 最大处理帧数（可选，默认300帧）

2. **选择视频文件**：
   - 点击"选择文件"按钮
   - 选择要测试的视频文件（支持MP4、AVI、MOV等格式）
   - 视频预览会自动显示

3. **提交检测**：
   - 点击"开始检测"按钮
   - 等待处理完成（会显示加载动画）

4. **查看结果**：
   - 检测完成后，结果会显示在页面下方
   - 包括：
     - 检测状态（confirmed/suspected/abnormal）
     - 置信度
     - 是否检测到动作
     - 检测到的药品列表
     - 动作时间线
     - 视频元数据
     - 处理耗时

## 测试建议

### 测试视频要求
- **格式**：MP4、AVI、MOV、MKV等常见格式
- **内容**：包含服药动作的视频
- **时长**：建议5-30秒
- **分辨率**：建议720p或以上
- **帧率**：建议25fps或以上

### 测试场景
1. **正常服药**：手拿药品送到嘴边
2. **疑似动作**：手接近口部但未完全到位
3. **异常情况**：没有检测到药品或动作

## 注意事项

1. **模型文件**：确保YOLOv8模型文件（ONNX格式）已放置在 `models/medication-intake.onnx`
2. **依赖安装**：确保已安装所有依赖（见 `pyproject.toml`）
3. **文件大小**：大视频文件可能需要较长的处理时间
4. **浏览器兼容性**：建议使用Chrome、Firefox或Edge浏览器

## 故障排查

### 问题：页面无法加载
- 检查服务是否正常启动
- 检查端口8000是否被占用
- 查看控制台错误信息

### 问题：视频上传失败
- 检查视频文件格式是否支持
- 检查文件大小是否过大
- 查看浏览器控制台错误信息

### 问题：检测失败
- 检查模型文件是否存在
- 查看服务端日志
- 检查视频文件是否损坏
