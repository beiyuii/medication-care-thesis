## ✅ 已对接接口

### 1. POST /api/logs/images - 上传日志图片

**功能描述**：上传服药事件的关键帧图片（摄像头截帧），图片将存储到本地/logs目录

**实现位置**：
- 服务层：`src/services/logService.ts` - `uploadLogImage(file, eventId)`

**功能特性**：
- ✅ 文件类型验证（仅支持 JPG、PNG、WEBP）
- ✅ 文件大小验证（≤2MB）
- ✅ 使用 `multipart/form-data` 格式上传
- ✅ 返回图片访问URL

**使用示例**：
```typescript
import { uploadLogImage } from '@/services/logService'

const file = // File 对象
const eventId = 123
const imageUrl = await uploadLogImage(file, eventId)
```

---

### 2. GET /api/logs/images - 查询事件图片列表

**功能描述**：根据服药事件ID查询该事件关联的所有关键帧图片

**实现位置**：
- 服务层：`src/services/logService.ts` - `fetchLogImages(eventId)`

**返回数据**：
- `id`: 图片ID
- `eventId`: 事件ID
- `url`: 图片URL
- `ts`: 上传时间戳

**使用示例**：
```typescript
import { fetchLogImages } from '@/services/logService'

const eventId = 123
const images = await fetchLogImages(eventId)
```

## 📝 注意事项

1. **文件上传格式**：
   - 必须使用 `multipart/form-data` 格式
   - 字段名：`file`（文件）、`eventId`（事件ID）

2. **文件限制**：
   - 支持格式：JPG、PNG、WEBP
   - 最大大小：2MB
   - 超出限制会抛出错误

3. **URL 处理**：
   - 后端返回的 URL 可能是相对路径或完整URL
   - 使用 `resolveImageUrl` 函数统一处理
   - 相对路径会自动拼接基础URL

4. **UI 集成**：
   - `DetectionRoomView.vue` 需要在检测到事件时调用 `uploadLogImage`
   - 上传成功后，将返回的 URL 保存到事件数据中
   - `HistoryCenterView.vue` 已支持显示图片，无需额外修改
