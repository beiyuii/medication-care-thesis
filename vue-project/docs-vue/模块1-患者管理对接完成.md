## 数据映射处理

### 后端 → 前端映射

1. **ID 类型**：后端返回 `number`，前端转换为 `string`（兼容现有代码）
2. **planStatus 映射**：
   - 后端：`active` / `paused`
   - 前端：`on_track` / `paused`
   - 映射规则：`active` → `on_track`，`paused` → `paused`
3. **时间格式**：后端返回 ISO8601（如 `2024-11-15T08:30:00Z`），前端显示为 `HH:mm`（如 `08:30`）
