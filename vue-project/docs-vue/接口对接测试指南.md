## 常见问题

### Q1: 登录后白屏
**原因**：可能是角色路由配置问题  
**解决**：检查控制台错误，确认路由配置正确

### Q2: Token 持久化不生效
**原因**：localStorage 可能被禁用  
**解决**：检查浏览器设置，允许网站使用 localStorage

### Q3: 接口调用失败
**原因**：后端未启动或 CORS 配置问题  
**解决**：
- 确认后端服务运行在 8080 端口
- 检查后端 CORS 配置是否允许前端域名

### Q4: 刷新页面后提示 401
**原因**：Token 过期  
**解决**：重新登录，获取新 Token
